# Atividade I - Programação para Internet - UNOESTE

## Introdução

Projeto de loja de cursos mockada: é possível acessar a home com listagem de cursos, formulário de login, formulário de cadastro, página de teste de erro e página de erro não interno publicamente e, com usuário logado, acessar página de detalhes de um curso, fazer a reserva/compra de vagas no curso e logout.

## Tecnologias

### Dependências

- `express`
- `express-session`

### Front-end

Utiliza HTML, CSS e JavaScript com funcionalidade mais recentes, por isso, recomenda-se o uso do Chrome.

- CSS: [`anchor-name`](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/anchor-name), [`anchor`](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/anchor), [`var`](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/var), [CSS Nesting](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Nesting/Using)
- JS: [Custom Elements](https://developer.mozilla.org/en-US/docs/Web/API/Web_components/Using_custom_elements)
- HTML: [`<template>` e `<slot>`](https://developer.mozilla.org/en-US/docs/Web/API/Web_components/Using_templates_and_slots)

## Rodando o Projeto

### Versões

Versão do Node: [22.14.0](https://nodejs.org/pt-br/download)
Gerenciador de pacotes: [PNPM 10.29.3](https://pnpm.io/pt/installation)

### Comandos

**Instalação de pacotes**: `pnpm i`
**Rodar o projeto**: `pnpm start` - roda na porta 3004
